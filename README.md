## Project Description
This project is done as part of the Digital Literacy course. The purpose of this project is to understand how digital tools and platforms are used by students in their daily academic work and why it is important to use them properly.

In this project, I completed different activities like designing an infographic, creating my profiles on online platforms, trying a coding platform, writing formal emails, and learning about cybercrime. These tasks helped me get a better idea of digital awareness and online safety.
